# Optional AntMaze Port Spec

AntMaze is the easiest existing game to bring into Chuds Arcade later.

## Arcade mode

Add:

- Daily Maze
- fastest escape leaderboard
- seeded share URLs
- monochrome thumbnail/card adaptation
- achievements

## Scoring option

For a clean leaderboard, use time as the primary metric:

```txt
primary: fastest completion
secondary: most crumbs collected
tertiary: fewest deaths
```

## Acceptance criteria

- AntMaze mounts through `ArcadeRuntime`.
- Daily seed is generated by the platform.
- Run submission uses normalized `RunResult`.
- Visual treatment is simplified to match the monochrome arcade shell.
